Shape Flow Commands & Gestures

👋 Two-Hand Gestures

  • Both open palms: Select next object
  • Both fists: Duplicate object
  • Left fist + right index up: Start voice listening
  • Both hands peace sign: Lock app
  • Both index fingers up: Start demo mode
  • Left palm + right fist: Delete object
  • Left fist + right palm: Save scene

🔄 Rotation Controls

  • Left 3 fingers up: Open rotation menu (1=X, 2=Y, 4=Z)
  • Right open palm: Move left/right to rotate
  • Right fist: Stop rotating

☝️ One-Hand Gestures

  • Open palm: Camera mode
  • Closed fist: Move object
  • Thumbs up: Make object bigger
  • Thumbs down: Make object smaller
  • Two fingers up: Zoom in
  • Three fingers up: Zoom out
  • Left index + middle up: Reset camera

💡 Easy Tips

  • Keep your hands steady & stay in good lighting.
  • Make sure object is selected before moving/scaling.
  • If it fails, lower your hand and try again slowly.

🎙️ Voice Commands

  • Create: "Add cube/sphere", "Build a spaceship", "Load school"
  • Select/Edit: "Select cube", "Delete that", "Duplicate", "Move up", "Scale down", "Rotate left"
  • Color: "Red", "Blue", "Green", "Gold", "White", "Black"
  • Camera: "Center cam", "Zoom in", "Reset camera"
  • App: "Save scene", "Clear scene", "Undo", "Start demo", "Lock app", "Stop listening"
